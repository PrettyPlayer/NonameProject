Создать приложение "Timer", которое должно уметь:
1. При запуске считывать время из файла time.txt или создавать файл time.txt по шаблону, если был не найден/пустой/нарушен синтаксис записи
2. Вести подсчет времени, при нажатии кнопки Play
3. Выводить время на экран
4. Приостанавливать подсчет времени, при нажатии на кнопку Pause
5. Воспроизводить звуки при нажатии на кнопки
6. Название времени должно склоняться от времени (1 секунда, 2 секунды..)
7. При закрытии приложения (Esc/Крестик) записывать результат в файлы README.md (для отображения на GitHub) и time.txt (для дальнейшей работы с приложением)

Создать приложение-синтезатор на языке Python с использованием библиотеки Pygame.
Приложение должно иметь:
1. Простой и понятный интерфейс
2. Лёгкую навигацию в меню с помощью стрелочек на клавиатуре
3. Выбор различных инструментов для игры
4. Регулировку громкости звука
5. Выбор времени затухания звуков синтезатора
6. Возможность включать/выключать ранее загруженные ноты, а также листать их
7. Управление синтезатором должно осуществляться на клавиатуре с помощью первых двух рядов букв
8. Набор различных ударных музыкальных инструментов
9. Управление ими должно осуществляться с помощью цифр NumPad
